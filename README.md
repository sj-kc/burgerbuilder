# Burger builder app

* React.
* Redux.
* React-Router.
* Webpack.
* Firebase.
* Authentication.
* ES6/ES7.
* Css modules.
* Hooks.

### Clone the repo

```bash
git clone https://github.com/sungkimcastro/burgerbuilder
```

### Install dependencies

```bash
yarn install
```

### Start a local server to preview the site

```bash
yarn start
```
